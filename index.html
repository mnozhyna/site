Signature Generator

Це генератор підписів для фонду «Множина».
Заповніть поля нижче, за потреби змініть розміри шрифтів, а потім натисніть кнопку Generate, щоб створити свій підпис для email.

Вводити тільки першу частину email до @
Налаштування шрифтів

Виділіть мишкою згенерований підпис та вставте його в налаштування підпису в Gmail згідно інструкції нижче.

Як додати підпис у Gmail

  1. Відкрий Gmail
  2. Натисни Settings (Налаштування) ⚙️ у правому верхньому куті
  3. Натисни See all settings (Переглянути всі налаштування)
  4. Прокрути сторінку до розділу Signature (Підпис)
  5. Натисни Create new (Створити новий)
  6. Введи назву підпису
  7. Встав підпис у поле редактора
  8. У розділі Signature defaults (Типові підписи) обери цей підпис для нових листів і відповідей
  9. Прокрути вниз і натисни Save Changes (Зберегти зміни)

Після вставки перевірте підпис, створивши новий лист.
